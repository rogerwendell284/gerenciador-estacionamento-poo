# gerenciador-estacionamento-poo
Projeto desenvolvido em Java com foco na aplicação prática dos conceitos de Programação Orientada a Objetos (POO). O sistema simula um gerenciador de estacionamento, permitindo o cadastro de veículos, cálculo de tarifas, controle de entrada e saída, além de acompanhar a capacidade total do estacionamento.
